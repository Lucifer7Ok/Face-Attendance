# Face Recognition Attendance System

A real-time face recognition attendance system built with Python and OpenCV.

## Features

- Real-time face recognition
- Automatic attendance recording
- Face encoding generation
- Image preprocessing and face cropping

## Tech Stack

- Python
- OpenCV
- NumPy
- face_recognition

## Project Structure

```text
src/      # Source code
data/     # Images and encodings
assets/   # UI resources
```

## Installation

```bash
pip install -r requirements.txt
```
